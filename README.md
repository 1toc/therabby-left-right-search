# Therabby Left-Right Search

左右に配置されたターゲットを探して選ぶ、視覚探索の試作課題です。

## Purpose

このアプリは、左右空間への探索の偏り、見落としの傾向、探索順序を観察し、臨床場面での説明や教材化を補助するための試作ツールです。

## Important Notice

このアプリは診断・判定を目的としたものではありません。  
半側空間無視や視空間認知障害の有無を単独で判断するものではなく、標準化された検査、ADL場面での観察、他職種の情報とあわせて総合的に解釈してください。

## Usage

GitHub Pagesで公開し、ブラウザ上で実行できます。
